Projects

Traffic Collisions in California

Used SQL, to analyze data released by California State Patrol (SWITRS). Goal of the analysis was to determine how many accidents are caused by the influence of alcohol, or due to inattention (such as using a cell phone to text or talk to others), and when these types of accidents tend to occur. The results were communicated through data visualization using Tableau

Bay Wheels User Analysis

Lyft purchased its bike share program from Ford (who owned GoBike) and they wanted to increase memberships in its rideshare program and needed to determine how their users, both past and present, use their product. SQL was used to solve these business-related problems. The JOIN statement and other features were used in SQL to combine tables together and investigate the differences between Lyft users and Ford users.

Grammys Project

The Recording Academy decided to split its website into two separate websites: grammy.com and recordingacademy.com. In this project, real data from both websites owned by The Recording Academy was analyzed using Python and Pandas. The purpose of the project was to examine the impact of splitting up the websites and analyze the data for a better understanding of the different trends and audience behavior on both sites. The results were visualized using Python's Plotly Express library

Prediction Model for team wins(MLB)

Utilized SAS to create the best 2-variable and 3-variable model to predict the number of wins for the Toronto Blue Jays. The best two-variable model to predict the number of team wins included the number of hits and the batting average as the independent variables and the best three-variable model to predict the number of wins for the Toronto Blue Jays included hits, batting average, and RBI as the independent variables.

Arrival Delay Prediction Model

Created a single-regression model using SAS to predict the arrival delay time at the Houston airport. Data for flights during the month of October were analyzed. Different plots were created and five independent variables were analyzed concerning the arrival delay time. Based on the results it was concluded that the carrier delay was the most responsible for arrival delays at the Houston airport for the month of October.

Covid-19 Cases and Deaths Analysis

Utilizing Linux and awk I analyzed the number of covid cases and deaths in Arizona from 2020-February 2022. I determined the number of new cases and deaths recorded each day during this period. Using awk I calculated the average number of new cases and deaths for each month and week. The averages were plotted on a graph using MATLAB to portray the variations.

Olympics Medalist

Utilized Python to analyze a dataset that consisted info regarding past Olympic medalists from 1896-2016. The following questions were answered using python: Who are the youngest and oldest medalists of all time? Are there physical differences between Summer Olympics medalists and Winter Olympics medalists? Which country had the most number of medals won?
